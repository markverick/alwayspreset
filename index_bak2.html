Set 1

The Traveler

พรีเซทที่นักท่องเที่ยวควรมี ไม่ว่าจะเดินทางไปที่ไหน ในเมือง, ป่า, ภูเขา ทะเล ธรรมชาติสวยๆ หรืออยากโชว์ไฟล์สไตล์ ที่หรูหราน่าดึงดูดอย่างเช่น ในคาเฟ่, ถ่ายอาหาร, เครื่องดื่ม The Traveler ถือว่าเป็นพรีเซทเบสิกที่ควรจะมีไม่ว่าจะเป็นมือใหม่หัดถ่ายภาพ หรือว่าจะเป็นมืออาชีพ พรีเซทชุดนี้ควรมีติดตัว เพื่อเตรียมพร้อมทุกสถานการณ์ให้ได้ภาพที่สวย น่าดึงดูด

Set 2

All Day All Night Portrait

พรีเซทที่เปลี่ยนภาพที่ดูธรรมดาให้ดูสดใสมีความน่ารักออกแนวเกาหลี ญี่ปุ่น จะเป็นการแต่งภาพถ่ายส่วนบุคคลให้ดูที่ความหวาน สดใสไม่ว่าจะไปถ่ายที่ไหนก็แล้วแต่ เชื่อว่าหลายๆคนอยากเปลี่ยนลุคของตัวเองให้เหมือนกับดาราหรือศิลปินเกาหลีที่ชื่นชอบ แต่ลำพังแค่การถ่ายรูปไม่เพียงพอ เลยจำเป็นที่จะต้องมีตัวช่วยในการปรับอารมณ์ของรูปถ่ายให้มีความคล้ายมากขึ้น สำหรับคู่รักที่อยากเห็นแฟนหนุ่มตัวเองดูดีแบบอปป้า หรือแฟนสาวน่ารักแบบสดใส All day all night protrait สามารถแก้ไขปัญหาเรื่องนี้ให้คุณได้

Set 3

Around the World

พรีเซทสำหรับนักท่องเที่ยว เป็นส่วนที่เพิ่มเติมเติมมาจากพรีเซท The traveler ที่จะมีโทนให้เลือกหลากหลาย สีสันแปลกตามากขึ้นกว่าเดิมและไม่ซ้ำเดิม ถูกพัฒนามาเพื่อนักท่องเที่ยวตัวจริง ไม่ว่าจะไปทั่วประเทศ หรือทั่วโลก แน่นอนว่าเราต้องอยากมีคอลเลคชั่นรูปภาพเอาไว้เยอะๆ ซึ่งบางทีการถ่ายในสถานที่คล้ายๆกันเป็นโทนคล้ายๆกัน อาจจะทำให้ดูน่าเบื่อได้ โทนที่หลากหลายที่ถูกพัฒนาขึ้นมาถึง60โทน จะทำให้คุณสามารถแต่งภาพเก็บเป็นคอลเลคชั่นที่หลากหลายอารมณ์ ดูแล้วไม่ซ้ำเดิม ไม่จำเจ แต่มีความสดใหม่ทุกครั้ง ไม่ว่าจะเป็นคาเฟ่ ป่า เขา ทะเล ถ่ายภาพสตรีท หรือ กลางคืน เซตนี้ถือว่าจัดเต็มสำหรับสายเที่ยวตัวจริง

Set 4

Dark Vintage Film

พรีเซทที่ออกโทนดาร์คๆ วินเทจ สำหรับคนที่ชอบโทนออกมืดๆหรือย้อนยุคก็ควรมีเซทนี้ติดตัว เพราะรูปถ่ายธรรมดา ไม่ว่าจะเป็นภาพถ่ายเมือง บุคคล หรือธรรมชาติคุณก็สามารถทำให้มันกลายเป็นโทนมืดๆที่ต่างจากภาพถ่ายมืดๆธรรมดา แต่จะแฝงไปด้วยความ Luxury และกลายเป็นภาพถ่ายที่ดูมีราคาขึ้นมาทันที
นอกจากความดาร์คแล้ว สิ่งที่มาคู่กันคือความวินเทจและความเป็นคันทรี่ทำมาสำหรับคนที่ชอบถ่ายภาพสไตล์ย้อนยุค บ้าน/เมืองเก่าแก่ และความเป็นคันทรี่ก็เหมาะสำหรับสายคาวบอย หรือชอบผจญภัยในที่ที่ออกชนบท เซตนี้ถือว่าทำออกมาแบบจัดเต็มครบรส จบในที่เดียวสำหรับคนชอบความดาร์คและความวินเทจ
